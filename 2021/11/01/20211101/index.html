Hexo deploy Error Spawn failed解決方案 - Part2

Posted on November 1, 2021 本文总阅读量次

作業系統：macOS
機台：MacBook Pro

繼之前 Hexo deploy Error Spawn failed解決方案 - Part1 之後，我立馬又遇到困難導致寫的部落格無法deploy

這一次的error log

幾經操作後，終於解了我這兩個禮拜的困擾。原來這有可能是ssh設定的問題，可能等級太低，我發了一番工夫才debug成功。

如果你剛好遇到這個問題，請發漏以下步驟試看看
可以節省很多時間～

  1. 首先，進入terminal > 重新產出 ssh key

    Generating public/private ed25519 key pair.

  2. Enter file in which to save the key
    輸入密碼並儲存

  3. ssh key 已經創建完成，接下來先確認identity added in ssh folder

    Identity added: /Users/tiffanyli/.ssh/id_ed25519 (tiffanylee.tiffany@gmail.com)

Identity added

  1. 複製ssh key以便新增至Github

  2. 設定Github內的 ssh key, 先進入Github > Setting > SSH and GPG keys

  1. 設定完成後，重新hexo cl > hexo g > hexo d > 完成

設定完後就順利deploy文章到github page了～
在嘗試過程中可以再確認一下是不是因為git version的關係等等逐步排查。
如果這方法有解決你的問題，那就恭喜囉～
